Dynamically output the calendar year with a copyright symbol and editable text

== Description ==

Displays a dynamically updating calendar year for your copyright. Additional text can be edited and formatted using the native block editor controls for color and typography.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/copyright-year` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Can I edit the copyright symbol or year? =

Currently, no. At the moment, they will always show at the beginning of the block. You can edit the rest of the text.
